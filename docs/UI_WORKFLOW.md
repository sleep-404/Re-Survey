**What AI changes:** AI does the initial tracing. Officer reviews and corrects.

## Key UI Design Principles

### 1. AI as First Draft, Human as Editor
- AI generates initial boundaries
- Officer reviews, doesn't start from scratch
- Huge time savings (hours instead of days per village)

### 2. Conflict-Driven Workflow
- Don't show all 500 parcels equally
- Prioritize the 20-30 that have issues
- Let "good" parcels auto-approve in background

### 3. Always Show Context
- Side-by-side: AI detected vs ROR expected
- Real-time area calculation as they edit
- Percentage difference always visible

### 4. AI-Assisted Editing
- "Snap to bund" when editing boundaries
- "Suggest split line" when splitting parcels
- Make corrections easy, not tedious

### 5. Clear Audit Trail
- Who approved what
- What was changed
- Why (reason dropdown + notes)
